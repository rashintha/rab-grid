Responsive Grid Example
This example shows a responsive grid.

    "grid_config: RABGridConfig = {
    responsive: true
}"}}
Striped Grid Example
This example shows a striped grid.
{{
    "grid_config: RABGridConfig = {
    theme: {
        striped: true
    }
}"}}
Grid Color Theme Example
This example shows color themes of the grid.
{{
    "grid_config: RABGridConfig = {
    theme: {
        color_theme: RABGridColorThemes.Primary
    }
}"}}
Color Theme with Striped Grid Example
This example shows color themes of a striped grid.
{{
    "grid_config: RABGridConfig = {
    theme: {
        striped: true,
        color_theme: RABGridColorThemes.Dark
    }
}"}}
Grid with Hoverable Rows Example
This example shows color themes of a hoverable grid.
{{
    "grid_config: RABGridConfig = {
    theme: {
        hoverable: true,
        color_theme: RABGridColorThemes.Default
    }
}"}}
Grid with Borders Example
This example shows how to enable borders in the grid.
{{
    "grid_config: RABGridConfig = {
    theme: {
        border: {
            enable: true
        }
    }
}"}}
Grid with Border Colors Example
This example shows color themes of the borders of the grid.
{{
    "grid_config: RABGridConfig = {
    theme: {
        border: {
            enable: true,
            color_theme: RABGridBorderColors.Default
        }
    }
}"}}
Grid with Custom CSS Classes Example
This example shows how to add custom css classes to the grid.
{{
    "grid_config: RABGridConfig = {
    theme: {
        custom_classes: 'table-sm align-middle'
    }
}"}}
